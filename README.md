# COM202---Final-Project
###
Final project submission by Fidan Alasgarli
###
Research Question / Problem Statement
###
Problem: In a parking facility with limited space, how can we manage the entry and exit of vehicles efficiently without wasting memory or manually reorganizing the list every time a car leaves?
###
This project addresses the "First-In, First-Out" (FIFO) nature of a parking lot.
###
Objective of the project
###
The project aims to design and implement a Circular Queue-based parking management system in C++.
